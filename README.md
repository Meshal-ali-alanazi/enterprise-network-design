# Enterprise Network Design

Cisco Packet Tracer enterprise network design project with a LaTeX report,
topology diagram, and packaged project deliverables.

This repository documents a CEN-431 Computer Networks Lab project. It is kept
as a portfolio artifact for network planning, topology documentation, and
technical reporting.

## Project Scope

- Enterprise-style Packet Tracer network topology
- Routing, switching, and VLAN-oriented network design work
- Topology screenshot for report documentation
- LaTeX report source and compiled PDF output
- Packaged submission archive with project files

## Repository Contents

- `FINAL-PROJECT.pkt` - Cisco Packet Tracer topology file
- `packet_tracer_topology.jpeg` - topology screenshot used by the report
- `network_project_report.tex` - LaTeX source for the project report
- `report.pdf` - compiled PDF report
- `network_project_report.zip` - packaged report/project archive

## Tools

- Cisco Packet Tracer
- LaTeX
- PDF documentation workflow

## How To Review

Open `FINAL-PROJECT.pkt` in Cisco Packet Tracer to inspect the topology.

Open `report.pdf` to review the final report, or use
`network_project_report.tex` if you want to inspect the LaTeX source.

## Notes

This project is an academic networking lab deliverable. It is intended to show
network design documentation and Packet Tracer project organization, not a live
production deployment.
